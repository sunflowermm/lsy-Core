## 何时使用

用户要写邮件、回邮件、催办、道歉、通知、邀请函、周报邮件摘要。

## 输出结构

1. **主题行**（≤50 字，信息完整）
2. **收件人称呼**（未知则用「您好」）
3. **正文**：背景一句 → 要点列表 → 明确行动项（谁、何时、做什么）
4. **结尾**：礼貌收尾 + 署名占位 `[您的姓名/部门]`
5. 需要时附 **英文版** 或 **简短版（≤100 字）**

## 语气档位

| 场景 | 语气 |
|------|------|
| 领导 / 客户 | 正式、敬语 |
| 同事协作 | 简洁、直接 |
| 催办 | 事实 + 截止时间，避免指责 |

## 禁止

- 不杜撰附件、会议链接、未给出的数据
- 不代发；只输出可复制正文
