## 何时使用

对内：周报、3P、项目状态、事故通报、FAQ 批量答复、部门通知、领导一页纸。

## 3P 周报（Progress / Plans / Problems）

```markdown
# [团队/项目] 周报 YYYY-MM-DD

## Progress（本周完成）
- …

## Plans（下周计划）
- …

## Problems（风险与阻塞）
- … | 需要谁协助 |
```

## 项目进展

- 里程碑状态：🟢正常 / 🟡风险 / 🔴阻塞
- 本周交付 / 偏差原因 / 决策项（需拍板的一问）

## 事故报告

1. **影响**：时间、范围、用户侧现象
2. **时间线**：发现 → 定位 → 缓解 → 恢复
3. **根因**（5 Whys 简版）
4. **行动项**：负责人 + 截止日期
5. **对外口径**（若需要，另段）
