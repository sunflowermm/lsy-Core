## 何时使用

「给老板一页纸」「决策 memo」「电梯汇报」「董事会摘要」。

## 模板

```markdown
# [主题] — 决策简报
**建议**：[一句话推荐] | **需决策**：[是/否 + 截止日]

## 结论（3 行内）
…

## 背景（≤5 bullet）
- …

## 选项
| 选项 | 优点 | 缺点 | 成本/时间 |
| A | | | |
| B | | | |

## 风险与缓解
- 风险 … → 缓解 …

## 下一步（若批准）
1. … | 负责人 | 日期 |
```

## 原则

- **结论第一**：老板不翻页也能懂
- 数字带单位与口径
- 未决项单独列出，不藏段尾
- 附件细节放工作区文件，正文只链路径

## 关联

- 数据支撑 → `office-research` / `office-chart`
- 会后 → `office-meeting`
